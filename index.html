Protocol, bot -> server¶

The format of a bot -> server packet is as follows

All packets¶

Name

Length

Description

Packet Length

2 bytes

Specifies packet_length

Message Type

1 B

A byte specifying message type

Data

May vary

Content depends on message_type

Data¶

Message type 0x01 is used for chat in the assigned channel

Message 0x01¶

Name

Length

Description

Author

Until NULL character is reached

Author of the message, ascii-encoded

Author

Until end of packet is reached

Content of the message, ascii-encoded

Message type 0x02 is used for bans and mutes

Message 0x02¶

Name

Length

Description

Ban / mute

1 byte

If it’s 0x10, it is a ban. If it is 0x11, it is a mute

Banner

Until NULL character is reached

Discord username of the one who banned the player

Player

Until NULL character is reached

Provided username of banned player

Date

Until NULL character is reached

Date when the ban/mute ends. If it’s before the current date, then it’s permanent. The date is an ascii-encoded, ISO 8601 (YYYY-MM-DDTHH:MM:SS.mmmmmm) string

Reason

Until end of packet is reached

Reason provided, ascii-encoded
